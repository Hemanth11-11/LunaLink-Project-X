# Interface Control Document

The shared data interface is a pandas `DataFrame` generated by
`lunalink.environment.build_environment_table`.

## Environment Columns Consumed by Subsystems

| Column | Unit | Consumers |
| --- | --- | --- |
| `t_s` | s | EPS, TCS, ADCS, TT&C |
| `x_eci_m`, `y_eci_m`, `z_eci_m` | m | ADCS, plotting |
| `vx_eci_mps`, `vy_eci_mps`, `vz_eci_mps` | m/s | ADCS |
| `altitude_m` | m | validation, plotting |
| `sun_hat_x`, `sun_hat_y`, `sun_hat_z` | unitless | EPS, TCS |
| `solar_flux_w_m2` | W/m2 | EPS, TCS |
| `eclipse_flag` | boolean | EPS, TCS, plotting |
| `gs_range_m` | m | TT&C |
| `gs_elevation_rad` | rad | TT&C, validation |
| `gs_contact_flag` | boolean | EPS, TT&C, validation |
| `moon_range_m` | m | TT&C |
| `moon_occulted_flag` | boolean | TT&C |
| `earth_ir_flux_w_m2` | W/m2 | TCS |
| `albedo_flux_w_m2` | W/m2 | TCS |
| `b_eci_x_t`, `b_eci_y_t`, `b_eci_z_t` | T | ADCS |
| `atmospheric_density_kg_m3` | kg/m3 | ADCS |

All subsystem modules return `(timeseries_df, summary_dict)`.
