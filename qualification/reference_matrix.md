# Reference Matrix

This matrix grounds the preliminary formulas and subsystem scope in recognized
engineering references. It is not a certification bibliography.

| Area | Reference | Usage in this tool |
| --- | --- | --- |
| Systems engineering | NASA/SP-2016-6105 Rev 2, NASA Systems Engineering Handbook | Requirements, assumptions, verification structure |
| Model credibility | NASA-STD-7009B, Standard for Models and Simulations | Credibility evidence, uncertainty, sensitivity, validation, and acceptance readiness |
| Software engineering | NASA NPR 7150.2, NASA Software Engineering Requirements | Software life-cycle evidence and verification readiness |
| Software assurance | NASA-STD-8739.8B | IV&V and flight-qualified claim boundary |
| ECSS system engineering | ECSS-E-ST-10C | Requirements and interface-control tailoring |
| ECSS software assurance | ECSS-Q-ST-80C Rev.2 / Rev.1 | Software product-assurance tailoring and reusable/non-deliverable software control |
| ECSS software engineering | ECSS-E-ST-40C | Software engineering process tailoring and software requirements traceability |
| Orbit and frames | Vallado, Fundamentals of Astrodynamics and Applications | Keplerian elements, J2, ECI/ECEF, ENU geometry |
| Orbit modeling | Montenbruck and Gill, Satellite Orbits | Force model and frame-upgrade roadmap |
| EPS | NASA Small Spacecraft Technology SoA, Power chapter, 2026 | Power generation/storage/distribution context |
| Thermal | NASA Small Spacecraft Technology SoA, Thermal Control chapter, 2026 | Absorbed/stored/generated/dissipated heat framing |
| ADCS/GNC | NASA Small Spacecraft Technology SoA, GNC chapter, 2026 | Smallsat actuator and attitude-control context |
| TT&C | NASA Small Spacecraft Technology SoA, Communications chapter, 2026 | Small spacecraft communications context |
| RF link budgets | JPL DESCANSO Design and Performance Summary Series | FSPL, C/N0, Eb/N0, margin accounting |
| RF standards | CCSDS Radio Frequency and Modulation Systems recommendations | Communications terminology and coding/modulation upgrade path |
| Earth-space propagation | ITU-R P.618 | Rain/atmospheric attenuation upgrade path for X-band ground links |
| Thermal constants | NIST CODATA Stefan-Boltzmann constant | Thermal radiation constant |
| Thermal design | Gilmore, Spacecraft Thermal Control Handbook | Lumped thermal balance, coating, radiator, and heater design context |
| ADCS theory | Wertz; Markley and Crassidis; Sidi | Quaternion kinematics, Euler dynamics, magnetic torques |
| Mission analysis tools | GMAT, Orekit | Independent orbit/contact/eclipse cross-check references |
| Spacecraft simulation architecture | Basilisk/Vizard documentation and Schaub-group publications | Future high-fidelity GN&C architecture and 3D visualization benchmark |
