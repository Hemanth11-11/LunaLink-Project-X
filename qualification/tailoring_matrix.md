# NASA/ECSS Tailoring Matrix

The project is structured to be compatible with later NASA/ECSS-style evidence,
but no certification is claimed.

| Guideline area | Applied now | Deferred for certification |
| --- | --- | --- |
| Requirements baseline | Implemented as a lightweight baseline | Formal stakeholder-approved requirement set |
| Interface control | Implemented for dataframes and subsystem contracts | Controlled ICD with versioned change board |
| Verification planning | Implemented with tests and VCD | Independent IV&V plan and witness records |
| Software assurance | Tests and ruff checks | Full NASA-STD-8739.8B assurance evidence |
| System engineering | Traceable architecture and assumptions | Full NASA SE lifecycle package |
| ECSS software product assurance | Testable Python package and limitations | ECSS-Q-ST-80C process evidence |
| ECSS system engineering | Requirements and assumptions registers | ECSS-E-ST-10C compliant review gates |
| Configuration management | Evidence bundle and deterministic seeds | Controlled release tags and signed baselines |
| External validation | Scenario exports created | Formal GMAT/Orekit numerical comparison |
| Acceptance | Not applicable | Authority acceptance after review |
