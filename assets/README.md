# LunaLink Visual Assets

`lunalink_spacecraft_model.stl` is a deterministic concept model generated by
`code/build_visual_assets.py`.

The model is inspired by the Project X LunaLink brief artwork: a gold
MLI-wrapped central bus, deployable blue solar-array wings, radiator panel,
high-gain dish antenna, and low-gain UHF antennas. Units are meters.

The STL is intended for visualization, presentation, and repository evidence. It
is not a manufacturing drawing, structural model, or CAD authority source.

## textures/

`textures/earth_bluemarble.jpg` and `textures/moon.jpg` are real equirectangular
surface maps used by the dashboard's 3D globe (`code/viz3d.py`). They are for
visualisation only. See `textures/CREDITS.md` for sources and licences. If the
files are absent the globe falls back to a procedural sphere.
